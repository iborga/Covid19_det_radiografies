Sistema automàtic per la detecció de COVID-19 en radiografies¶

Una de les applicacions de AI és el processament de imatges mèdiques. En particular, l'ús de les radiografies per una detecció no invasiva i ràpida del Covid-19 pot resultar extremadament útil per ajudar i agilitzar la tasca del personal mèdic [1][2].

En aquest projecte desenvoluparem un detector Deep Learning de Covid-19 en radiografies. Per això, utilitzarem imatges de la base de dades "Covid-chestxray-dataset" [3], generada per uns investigadors del grup de recerca Mila i de la Universitat de Montreal [4]. També farem servir imatges de radiogradies de pacients sans i amb pneumonia bacterial extretes de la competició de Kaggle "Chest X-Ray Images (Pneumonia)" [5].

En total, disposem d'un total de 426 imatges, dividides en conjunts d'entrenament (339 imatges), validació (42 imatges) i test (45 imatges). Les particions es donen en llistes ".txt", on a cada imatge se li assigna una etiqueta:

  • 0) Healthy
  • 1) Covid-19
  • 2) Pneumonia

L'objectiu consisteix en desenvolupar un sistema basat en xarxes neuronals capaç de classificar correctament les imatges en aquestes 3 categories.

Nota: Els resultats obtinguts per models entrenats en aquesta base de dades són purament per finalitats educatives i no es poden fer servir per un diagnòstic real sense validació clínica.

def datf(file,size):
    ftrain= open(file, 'r')
    ftrain= ftrain.read()
    ftrain= ftrain.split('\n')
    case=[]
    label=[]
    for i in ftrain[:-1]:
        c= i.split('\t')
        img= Image.open('DATASET/'+c[1]).convert('L') # mode luminositat (una sola dimensió en profunditat)
        img= np.array(img.resize(size))
        case.append(img)
        label.append(int(c[0]))
    return(pd.DataFrame({'Case': case, 'Label': label}).sample(frac=1,random_state=42).reset_index(drop=True))

# Gata augmentation mitjançant combinació de rotate(lleuger, entre 4-8 graus) i crop.
# Girarem totes les imatges cap a dreta i esquerra abans de fer el zoom, triplicant així les mostres per train.
# Finalment no farem servir flip per no donar caracteristiques errònies al model (el cos humà no és simètric).

def rota(img,ang):
    return(Image.fromarray(img).rotate(ang))
def flip(img):
    return(Image.fromarray(img).transpose(Image.FLIP_LEFT_RIGHT))
def crop(img):
    return(Image.fromarray(img).crop((round(size[0]*0.05), round(size[1]*0.05),
                                      round(size[0]*0.95), round(size[1]*0.95))).resize(size))

ang=4
dataug=pd.DataFrame(columns=['Case','Label'])
for i in traindf.index:
    img= np.array(crop(np.array(rota(traindf.loc[i,'Case'],ang))))
    dataug= dataug.append(pd.DataFrame([[img, traindf.loc[i,'Label']]],columns=['Case','Label']))
    img= np.array(crop(np.array(rota(traindf.loc[i,'Case'],-(12-ang)))))
    dataug= dataug.append(pd.DataFrame([[img, traindf.loc[i,'Label']]],columns=['Case','Label']))
    ang+=1
    if ang>8:
        ang=4

trainaug= traindf.append(dataug).sample(frac=1,random_state=42).reset_index(drop=True)

# Definim x i y per cada dataset generat

def x_y(df):
    enc= OneHotEncoder(categories='auto')
    y= enc.fit_transform(np.array(df.Label).reshape(-1,1)).toarray()
    x=[]
    for i in df.Case:
        x.append(np.array(i).ravel())
    x= np.array(x).reshape(len(df.index),size[0],size[1],1)
    return(x,y)

# Funció per plot del history.

def phistory(mod,fit):
    l=len(fit.history['loss'])
    plt.style.use('ggplot')
    plt.figure(figsize=(12, 4))
    
    plt.subplot(1, 2, 1)
    plt.plot(np.arange(0, l), fit.history['accuracy'], label='train_acc')
    plt.plot(np.arange(0, l), fit.history['val_accuracy'], label='val_acc')
    plt.xlabel('Epoch #')
    plt.ylabel('Accuracy')
    plt.legend(loc='lower right')
    
    plt.subplot(1, 2, 2)
    if fit.history['val_loss'][-1]>2:
        plt.ylim(0, 40)
    else:
        plt.ylim(0, 2)
    plt.plot(np.arange(0, l), fit.history['loss'], label='train_loss')
    plt.plot(np.arange(0, l), fit.history['val_loss'], label='val_loss')
    plt.xlabel('Epoch #')
    plt.ylabel('Loss')
    plt.legend(loc='upper right')
    plt.show()
    
    evm= mod.evaluate(x_val, y_val, verbose=1)
    print('Precisió del model:', evm[1],'\n')
    print('Pèrdua del model:', evm[0],'\n')

Explainability (1 punt)¶

Investiga sobre les tècniques que es fan servir per entendre les decicions d'una CNN. Per exemple, Els Class Activation Maps (CAM)[4] són una modificació en l'arquitectura d'una CNN de classificació de imatges que ens permet visualitzar quines parts de la imatge es consideren per classificar una classe amb una determinada etiqueta. Variacions de les CAMS s'han aplicat recentment per entendre el comportament de les xarxes per detectar Covid19[8].

Una altre mètode és el popular Deep Dream[11]. Aquest mètode, a part de generar imatges amb un estil bastant artístic/psicodèlic, serveix per entendre el tipus de característiques que una xarxa busca en la imatge d'entrada per classificar-la amb una determinada etiqueta.

Implementa alguna tècnica de visualització per intentar entendre les decisions de la CNN entrenada.

from keras import backend as K

def cam(model,img):
    predict= model.predict(img)
    target_class= np.argmax(predict[0])
    
    grads= K.gradients(model.output[:,target_class], model.layers[1].output)[0]
    pooled_grads= K.mean(grads,axis=(0,1,2))
    iterate= K.function([model.input], [pooled_grads, model.layers[1].output[0]])
    pooled_grads_value,conv_layer_output = iterate([img])
    
    for i in range(24):
        conv_layer_output[:,:,i] *= pooled_grads_value[i]
    
    heatmap = np.mean(conv_layer_output,axis=-1)
    
    for x in range(heatmap.shape[0]):
        for y in range(heatmap.shape[1]):
            heatmap[x,y] = np.max(heatmap[x,y],0)
    
    heatmap = np.maximum(heatmap,0)
    heatmap /= np.max(heatmap)
    return(heatmap)
